About

DozeAlert helps commuters sleep peacefully and arrive confidently. Whether you're on a train, bus, car, or rideshare, DozeAlert monitors your trip and wakes you before you miss your stop.

  • Location-based wake alerts with adjustable radius
  • Transit mode with stop countdown (GO, TTC, and GTFS agencies)
  • Background monitoring while you rest
  • Voice, vibration, and alarm alerts
  • Map picker and saved destinations
  • No account required — trip data stays on your device

Permissions & privacy

DozeAlert needs clear permission choices to monitor trips reliably. During first-time setup the app explains each step. For monitoring on Android, set:

  • Location: Allow all the time (not only while using the app)
  • Notifications: Allowed (ongoing trip notification and alerts)
  • GPS: Location services turned on
  • Battery: Unrestricted recommended on some phones

We do not show ads, sell personal data, or require an account. Map search uses Google Maps Platform; optional transit downloads use public open-data hosts. Read the full policy: Privacy Policy

Install from APK (sideload)

Prefer Google Play when available. Use a direct APK if you sideload or need a build before the store listing appears in your region.

  1. Download the APK using the button above. Wait until the download finishes (check your notification shade or Downloads folder).
  2. Open the downloaded file — tap the notification, or open Files / Downloads and tap dozealert-latest.apk.
  3. Allow installation if Android blocks it. When prompted, tap Settings and turn on Allow from this source for Chrome, Samsung Internet, or whichever app you used to download. Then go back and tap Install.
  4. Complete setup in the app — grant permissions as prompted (location all the time, notifications, GPS on). See Privacy Policy for details.
  5. Open DozeAlert, pick a destination, set your wake radius, and tap Start monitoring.
Tip: After installing, you can turn off “Install unknown apps” again for your browser. It only needs to be enabled during installation.
Android version-specific help

Android 13 and newer: Settings → Security & privacy → More security settings → Install unknown apps → select your browser → Allow.

Android 12: Settings → Apps → Special app access → Install unknown apps → your browser → Allow.

Samsung Galaxy: Settings → Biometrics and security → Install unknown apps → select the app you downloaded with → Allow.

“App not installed” error: Uninstall any older test build first, or download the APK again — the file may have been corrupted during download.
